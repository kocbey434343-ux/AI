
ENVIRONMENT SETUP GUIDE
======================

To configure your production environment:

1. Create a .env file in the project root
2. Add the following variables:

USE_TESTNET=false
ALLOW_PROD=true
BINANCE_API_KEY=[your_production_api_key]
BINANCE_SECRET_KEY=[your_production_secret_key]

3. Obtain Binance API credentials:
   - Go to https://www.binance.com/en/my/settings/api-management
   - Create new API key
   - Enable "Enable Spot & Margin Trading"
   - Copy API Key and Secret Key
   - Add to .env file

4. Security recommendations:
   - Add IP restrictions to API key
   - Use minimal required permissions
   - Store credentials securely
   - Never commit .env to version control

5. Verify setup:
   python simple_env_validator.py
